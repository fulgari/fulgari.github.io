关注

访问控制具体关注的是client和library creator之间的一种沟通communication

  1. 一个java文件里面只能有一个public的class
  2. .java文件编译之后后缀变成了.class，这就是一个可以工作的程序了。可以通过jar archiver打包变成
    JAR执行。和普通文件编译直接变成可执行文件或库library不一样。
  3. 这些乱七八糟的.java和.class通过package关键字笼络在一起，就像是一把伞把他们裹起
    来了。
  4. 所谓import其实就相当于将一个package引进来了。这两个关键字作用都是为了防止撞名name
    clashing。

2017年11月8日

Java

Java是一种静态的语言，我是后来才知道的。

初次

第一次接触Java是初一时，那时候手机刚刚能够连上GPRS上网。还记得那时候最流行的就是在拙劣的静态文本网页里面打开QQ空间偷菜、抢车位。我老早就期待可以像同学们一样用手机玩各种各样的手机游戏了。那时还没有很多软件市场，通过浏览器，我们可以在网上下载一个个的.jar包，也正是这些个.jar包，让我开始了对神秘的编码世界的好奇。

虽然后面这些日子里，我一直都没有思考过自己想要做什么，自己未来10年后打算过着怎么样的生活，自己怎么养活自己，我从来没有考虑过。因为我是一个及时行乐的人。

后来

从大四开始，我开始为生计发愁。所谓少年不知愁滋味，我可能也开始不再少年了吧。我不否认自己是一个不够潇洒的人。我只是一个蝼蚁罢了，一只多少有点抑郁的局外人。我希望能够从工作中多少找到些意义，如果说我不得不工作的话。当然，我必须工作。

其实

看来我除了发些牢骚，也没有什么有价值的东西写了。我今晚还得好好看看JSP呢。就酱！
